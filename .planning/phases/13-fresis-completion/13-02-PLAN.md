    **Step 1: Add route handlers in fresis-history.ts**

    1. `PUT /:id` route:
       - Extracts id from params, body from request
       - Calls deps.updateRecord(id, body)
       - Returns 200 with updated record or 404 if not found

    2. `POST /reassign-merged` route:
       - Extracts { recordIds, newMotherOrderId } from body
       - Validates: recordIds is non-empty array, newMotherOrderId is string
       - Calls deps.reassignMergedRecords(recordIds, newMotherOrderId)
       - Returns 200 with { updated: count }

    3. `POST /archive` route:
       - Extracts { recordIds } from body
       - Calls deps.archiveRecords(recordIds)
       - Returns 200 with { archived: count }

    4. `POST /discounts/bulk-upload` route:
       - Extracts discounts array from body (or multipart file)
       - Calls deps.bulkUploadDiscounts(discounts)
       - Returns 200 with { inserted, updated }

    **Step 2: Update FresisHistoryRouterDeps type**

    Add new deps to the interface:
    ```typescript
    type FresisHistoryRouterDeps = {
      // ... existing deps
      updateRecord: (id: string, updates: Partial<FresisHistoryRecord>) => Promise<FresisHistoryRecord | null>;
      reassignMergedRecords: (recordIds: string[], newMotherOrderId: string) => Promise<number>;
      archiveRecords: (recordIds: string[]) => Promise<number>;
      bulkUploadDiscounts: (discounts: DiscountEntry[]) => Promise<{ inserted: number; updated: number }>;
    };
    ```

    **Step 3: Wire deps in server.ts**

    Import new repository functions and bind pool:
    ```typescript
    updateRecord: (id, updates) => updateRecord(pool, id, updates),
    reassignMergedRecords: (ids, motherId) => reassignMergedRecords(pool, ids, motherId),
    archiveRecords: (ids) => archiveRecords(pool, ids),
    bulkUploadDiscounts: (discounts) => bulkUploadDiscounts(pool, discounts),
    ```

    **Step 4: Add route tests in fresis-history.spec.ts**

    Test each new endpoint with mocked deps:

    - POST /reassign-merged returns 200 with count
    - POST /reassign-merged returns 400 for invalid body
    - POST /archive returns 200 with count
    - POST /discounts/bulk-upload returns 200 with stats
